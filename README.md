# 📊 SQL Social Media Analysis

## 🚀 Project Overview

This project analyzes Instagram-style social media data using SQL to generate actionable marketing and engagement insights.

The objective was to simulate the role of a Data Analyst working with a marketing team to improve user engagement, retention, and growth through data-driven decision-making.

This project scored **90%** during evaluation.

---

## 🗂 Dataset Structure

The database consists of 7 relational tables:

- **users** – Registered user information  
- **photos** – User-uploaded content  
- **comments** – User comments on posts  
- **likes** – Engagement actions  
- **follows** – Network relationships  
- **tags** – Content categories  
- **photo_tags** – Tag-to-photo mapping  

---

## 🎯 Key Objectives

- Analyze user activity distribution  
- Measure engagement concentration  
- Evaluate hashtag performance  
- Identify influencers & brand ambassadors  
- Perform behavior-based user segmentation  
- Generate business-driven marketing recommendations  

---

## 🔎 Key Analysis & Insights

### 1️⃣ User Segmentation
Users were categorized into:
- Super Creators (6%)
- Active Users (37%)
- Low Activity Users (31%)
- Inactive Users (26%)

📌 Insight: **57% of users represent untapped growth potential**, indicating strong opportunity for re-engagement strategies.

---

### 2️⃣ Engagement Concentration
A small percentage of creators generate the majority of platform interactions.

📌 Insight: Engagement follows a concentration pattern driven by high-value creators.

---

### 3️⃣ Hashtag Performance
Lifestyle and emotion-driven hashtags showed the highest engagement efficiency.

📌 Insight: Content themes significantly influence interaction levels.

---

### 4️⃣ Posting Consistency vs Engagement
Users with higher posting frequency consistently achieved stronger engagement.

📌 Insight: **Consistency is a major driver of audience interaction.**

---

### 5️⃣ Influencer & Brand Ambassador Identification
Selection was based on:
- Engagement per post  
- Posting consistency  
- Community participation  

📌 Insight: Engagement efficiency is a stronger indicator of influence than follower count.

---

## 🛠 Technologies Used

- MySQL
- SQL (CTEs, Joins, Aggregations, Group By, Subqueries)
- Data Segmentation Techniques
- Business Strategy Modeling

---

## 📈 Business Recommendations

- Launch targeted reactivation campaigns for inactive users  
- Convert active users into high-value creators  
- Promote high-performing content themes  
- Use engagement-per-post as a core KPI  
- Develop structured influencer & ambassador programs
